# Create a new branch
```bash
git branch <branch-name>
```

# List all branched
```bash
git branch
```

# Switch to a branch
```bash
git checkout <branch-name>
```

# Delete a branch
## Switch to another branch
```bash
git branch -D <branch-name>
```

# Create a branch from another in one step
```bash
git checkout -b <branch-name>
```
